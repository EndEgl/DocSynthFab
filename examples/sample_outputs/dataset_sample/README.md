# Sample Output Dataset

This folder contains a small early-alpha sample output generated by DocSynthFab.

It is intended to show the generated dataset structure:

dataset_sample/
  images/   Generated document page images.
  gt/       Ground truth JSON files.
  ann/      Annotation JSON files.
  masks/    Segmentation masks, when available.
  reports/  QC and run-level metadata, when available.

These samples are not formal benchmark results. They are provided to demonstrate layout diversity, multilingual rendering, annotation structure, and dataset packaging.
