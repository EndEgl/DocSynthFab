# COCO Export

This export converts document block boxes to COCO layout detection annotations.

## Categories

1. plain_text
2. table_region
3. math_latex
4. figure

## Structure

- `images/train|val|test/`
- `annotations/instances_train.json`
- `annotations/instances_val.json`
- `annotations/instances_test.json`
- `categories.json`
